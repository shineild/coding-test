# 알고리즘 문제 정리
그동안 풀은 알고리즘 문제를 정리한 아카이브입니다.


## 정리된 코드는 3가지 유형이 존재합니다.
1. 해설을 보지 않고 스스로 푼 문제
2. 알고리즘 분류를 확이 후 연관 문제 풀이를 학습 후 푼 문제
3. 문제풀이를 보고 이해 한 문제


## 블로그에 작성한 문제으 유형으 다음과 같습니다.
1. 인상깊었던 스스로 푼 문제
2. 이전에 학습한 알고리즘 개념을 직접 적용해서 풀어본 문제
